How to add "Send with SYF" to the context menu:
- install the nautilus-actions package (e.g. apt-get install nautilus-actions)
- open the Nautilus-Actions configuration tool
- define a new action
- assign a value to the context label (e.g. "Send with SYF")
- optionally configure the tooltip and the icon
- move to the Command tab
- set the path pointing to the SYFPicker executable
- insert "%F" in the Parameters field
- test the new feature
